# BickerstaffEtAl_PCIRRStage1_2026
Repository for the manuscript, data and code related to the PCI RR Stage 1 submission "Objective and subjective assessment of light sensitivity in physiological responses to bright light in humans"

---

## Folder description

- Folder `code` contains all the R code needed to run the calibration of the light source, the modelling of the pilot data (OO-EMG), the BFDA, and the visualisations included in the report.
- Folder `R`contains a helper R script for easily accessing project-wide variable and functions.
- Folder `renv` contains files necessary for the `renv` package for manage the R environment
- Folder `data` contains all the necessary data to run all analyses and visualisations included in the report, i.e. processed pilot data and calibration measurements.
- Folder `outputs` contains all the outputs generated from the code and data mentioned above, including all figures included in the report, the OO-EMG model fit, the design matrix, and the BFDA results.
- Folder `visual_abstract` contains the visual abstract figure
- Folder `manuscript_stage1` contains the Stage 1 document (`.docx`)

- R version: **[4.4.0]**
- RStudio

All R package dependencies are managed using [`renv`](https://rstudio.github.io/renv/).

## Reproducing the analyses

### 1. Clone this repository

```bash
git clone https://github.com/tscnlab/BickerstaffEtAl_PCIRRStage1_2026.git
```

### 2. Open the R project

Open `BickerstaffEtAl_PCIRRStage1_2026.Rproj` in RStudio (or set the working directory to the project root).

### 3. Restore the R environment

```r
install.packages("renv")
renv::restore()
```

### 4. Run the `.Rmd` scripts

Running the code will recreate:

- The OO-EMG LMM fit object and model output
- The design matrix
- The BFDA results
- Figures in outputs/figures/

We recommend running the code in that order: `calibration`, `modelling`, `bfda`, and `visualisation`, as some scripts depend on outputs generated by other scripts.
