# Data description

## `main_data`: EMG data for modelling

- `emg_outcomes_trial.rds` includes trial-level average EMG responses for each light condition and each muscle. This data is used for fitting the model used for the sensitivity analysis.

## `average_traces`: Group-level average traces and ratings for visualisation

- `emg_pilot_data_gavg.csv` includes the group-level average OO-EMG response trace with SD. `time_sec` is time relative to light onset.
- `pupil_pilot_data_gavg.csv` includes the group-level average pupillary response trace with SD. Sample rate is 60 Hz.
- `ratings_pilot_data_gavg.csv` includes the group-level average rating for visual discomfort with SD, for each light condition.

## `data_loss`: Data loss in the datasets

- `emg_pilot_data_loss.csv` includes the EMG data loss for each muscle. This is defined as the proportion of all datasets for this muscle that are not included in the analysis (due to electrodes detaching, or incomplete data due to technical problems).
- `pupil_pilot_data_loss.csv` includes the pupillary data loss for each participant and light condition. This is defined as the proportion of values whose associated confidence rating falls below 0.7 (0 is no confidence, 1 is total confidence).

## `preprocessing_demos`: Example participant data for running preprocessing scripts

This folder contains:
- Two demo R-readable datasets (`demo_pilot_emg_data.rds` and `demo_pilot_pupil_data.rds`) to test the EMG and pupillometry data preprocessing pipelines,
- Associated metadata (`metadata.csv`).

## `light_source_calibration`: Light source calibration data

All calibrations were performed using the Spectraval 1511 spectrometer from JETI Technische Instrumente GmbH, Jena, Germany. The light source used was the SpectraTune Lab (Ledmotive Technologies S.L., Barcelona, Spain).

The calibration data are organised into four folders that reflect the processing workflow:

* `1_raw/` contains the original spectral measurement files exported from the spectrometer. These files retain the original measurement structure and are named with the suffix `_raw`.
* `2_preprocessed/` contains the same calibration files after preprocessing. These files are named with the suffix `_preprocessed` and were prepared for subsequent processing in [luox.app](https://luox.app/).
* `3_luox_outputs/` contains the outputs generated by [luox.app](https://luox.app/) from the preprocessed spectral data. These files are named with the suffix `_luox`.
* `4_outcomes/` contains the final summarised calibration outcomes used for analysis and reporting. The necessary outcomes are summarised in `1_calibration_reference_outcomes.csv`.

The raw calibration files are as follows:

* `1_calibration_reference_raw.xlsx` contains the SpectraTune Lab light source calibration data. Each condition was sampled 15 times, i.e. for each of the 10 LEDs as well as for white light, where all LEDs were set to the same intensity. Measurements were made at the following settings: 25, 50, 100, 200, 400, 800, 1200, 1600, 2000, 2400, 2800, 3200, 3600, 4000, and 4095/4095. The final, 166th measurement is the spectral measurement obtained when the light source was off, corresponding to total darkness.
* `2a_target_settings_raw.xlsx` contains the spectral measurements for the calculated primary settings corresponding to target irradiance values. Measurements were made for all six primaries used, across four lab visits, with six repetitions per measurement: the first three in forward order, from primary 2 to primary 7, and the last three in reverse order, from primary 7 to primary 2. This resulted in 144 total measurements. Unlike the reference measurements, these measurements were made with the infrared camera on, which produced additional radiation in the infrared range. The calculated settings were slightly off target and were therefore manually corrected.
* `2b_target_settings_corrected_raw.xlsx` contains the spectral measurements for the corrected primary settings corresponding to the target irradiance values. The measurement structure is the same as for `2a_target_settings_raw.xlsx`.
* `3_pilot_settings_raw.xlsx` contains the spectral calibration file for the pilot light conditions. The spectra were kept constant at a target CCT of 5000 K. A 0.3 ND filter was used for these calibration measurements, as well as for the actual experiment. Only one round of measurements was made, resulting in six total measurements, one for each light condition.

The raw files were preprocessed before being processed in [luox.app](https://luox.app/). The resulting luox outputs are retained for transparency, while the final outcome variables required for analysis are provided in `4_outcomes/1_calibration_reference_outcomes.csv`.
